How to get all references for the SCP:SL Axon Client:

1. Install and *Start* the Axon Client. Melonloader will generate all files you will need.
2. Go to your axon client installation and locate the `Melonloader` folder
3. Copy the Folders Il2cppAssemblies,Managed and net6 into this Refs directory
4. Go back to your Axon client Installation and navigate to UserLibs
5. Copy BouncyCastle.Cryptography.dll into this Ref directory
6. You are done
